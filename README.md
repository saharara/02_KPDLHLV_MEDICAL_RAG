# 02_KPDLHLV_MEDICAL_RAG
Xây dựng mô hình ngôn ngữ nhỏ SLM tiếng Việt cho dữ liệu y tế sử dụng phương pháp RAG (Retrieval-augmented generation).
## Mô tả
Thiết kế mô hình RAG nhằm trả lời câu hỏi y khoa dạng Đúng/Sai dựa trên tri thức y khoa tiếng Việt chuyên ngành, với mục tiêu giảm hallucination của các mô hình ngôn ngữ và cung cấp câu trả lời minh bạch dựa trên tài liệu y tế chính thống.
Dữ liệu sử dụng trong mô hình được lưu trữ tại nền tảng kaggle: kaggle.com/datasets/nguynhonghanh/kpdlhlv
